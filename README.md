### 环境要求

- [其他必要环境]

### 安装步骤

1. 克隆项目仓库
   ```bash
   git clone <repository-url>
   ```

2. 进入项目目录
   ```bash
   cd WesleyCode
   ```

3. 恢复 NuGet 包
   ```bash
   dotnet restore
   ```

4. 构建项目
   ```bash
   dotnet build
   ```

5. 运行项目
   ```bash
   dotnet run
   ```
